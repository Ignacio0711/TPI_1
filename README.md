# <center>Programación de APP para dispositivos móviles</center>

## **1. ¿A que nos referimos cuando hablamos de APP?**
#### Grupos de programas que están orientado a un usuario final.

## **2. ¿Que es el frontend y que es el backend?**
#### **Frontend** es la parte que interactúa con el usuario, y el **backend** es la información que se encuentra detrás de la interfaz, lo que no se ve. Se comunican por medio de API.

## **3. ¿Ques es la API REST?**
#### Es un estándar que permite la comunicación entre en el front y el back.

## **4. ¿Qué es el formato JSON?**
#### Un formato JSON es un formato de texto que funciona como almacenamiento
#### [
#### {“Alumno”: “Juan”;
####  “Apellido”: “Pérez”;
#### “notas”:[8,9,8];
#### }
#### ]    

## **5. ¿Que es una libreria (o biblioteca) y que es un framework?**
#### Una librería o biblioteca es un conjunto de funciones o clases enfocados en una tarea que se puede usar las veces que sea. Un framework es la estructura predefinida que proporciona la base para construir o como organizar el código.

## **6. ¿Que es un sistema de control de versiones?**
#### Un sistema de control de versiones es una herramienta de software que registra los cambios realizados en archivos o códigos a lo largo del tiempo. Sirve para gestionar el historial de modificaciones, permitir el trabajo colaborativo simultáneo sin sobrescribir el trabajo ajeno, revertir errores y respaldar proyectos de manera segura.

#### En el caso de que dos desarrolladores modifiquen la misma línea de código, prevalece la versión del administrador del proyecto.

## **7. ¿Que es la Programacion Orientada a Objetos?**
#### La programación orientada a objeto es un paradigma, una forma de resolver un problema.

## **8. ¿Cual es las propuestas de los modificadores públicos y privados?**
#### El modificador private permite que los elementos solo sean visibles en la propia clase. En cambio, public los elementos son visibles y accesibles desde cualquier otra clase del proyecto.

## **9. Explique las principales diferencias entre los sistemas operativos de Android y Iphone.**
#### Android: su hardware funciona en diversas marcas, utiliza el lenguaje de Java, su tienda de aplicación es Play Store.
#### iOS: únicamente funciona con dispositivos Apple, el lenguaje que utiliza es Swift, y su tienda de aplicación es App Store.
